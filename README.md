# date-cli

## About <a name = "about"></a>

Get current iso time string for command line. 

## Usage <a name = "usage"></a>

```bash
date-cli
// echo 2021-06-19T19:44:34.501Z
```

### add

```bash
date-cli add --from now --months 3
```

### sub

```
date-cli sub --from monthBeginning --years 1
```
